## Overview
**Riona Jewels** is a front-end jewelry brand website showcasing featured collections, products, and brand details.  
The project focuses on clean UI design, responsive layout, and modern styling suitable for an e-commerce landing page.

This project was built to practice:
- Responsive web design
- Component-based UI layout
- Bootstrap integration
- Clean CSS structuring

---

## Features
- Responsive navigation bar with scrollspy
- Hero section with overlay and call-to-action
- Featured products section
- Full product collection grid
- About section with brand highlights
- Contact section (phone, email, address)
- Fully responsive across mobile, tablet, and desktop
- Hover effects and smooth scrolling

---

## Tech Stack
- **HTML5**
- **CSS3**
- **Bootstrap 5.3**
- **Font Awesome**
- **JavaScript (basic setup)**
